# JWT-Auth
Authenticating and refreshing Tokens Using JWT


## Functionalities

- Login
- Logout
- Token Refresh


## Used Stack

**Front-end:** React

**Back-end:** Node, Express


## Instalation

Install api with npm

```bash
  cd jwt-auth-api 
  npm install
```

Install web client with npm

```bash
  cd jwt-auth-webClient 
  npm install
```
